# 💳 Loan Prediction — Kaggle Project

This repository contains my solution to a Kaggle competition focused on **predicting loan approval status** using customer data.

---

## 🎯 Objective

The goal of this project is to build a **machine learning classification model** that predicts whether a loan application will be **approved or rejected**, based on applicant information such as income, credit history, and demographic features.

---

## 📊 Dataset

The dataset consists of two main files:

* **`train.csv`**: Training data including the target variable
* **`test.csv`**: Test data used for generating predictions

### 🔍 Features include:

* Applicant information (gender, marital status, dependents)
* Financial data (applicant income, co-applicant income)
* Loan details (loan amount, loan term)
* Credit history
* Property area

---

## 🧠 Methodology

The project follows a standard data science pipeline:

### 🔹 1. Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature transformation and scaling

### 🔹 2. Modeling

* Training classification models (e.g., Logistic Regression, Decision Trees)
* Comparing model performance

### 🔹 3. Evaluation

* Using classification metrics (accuracy, etc.)
* Model selection and basic tuning
